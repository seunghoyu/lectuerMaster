# 추가된 기능 목록

## 📅 업데이트 날짜: 2024년

---

## ✨ 주요 기능

### 1. **강의리스트 저장 기능**
- 선택한 강의를 개인 리스트로 저장
- 리스트 이름 지정 가능
- 페이지당 항목 수 설정 (10~1000개)
- 선택 상태 유지 (페이지 이동 시에도)

### 2. **저장된 리스트 관리**
- 사이드바에서 "내 강의리스트" 드롭다운으로 확인
- 개별 리스트 클릭 시 해당 리스트만 조회
- 리스트 이름 수정
- 리스트에 강의 추가
- 리스트 삭제

### 3. **공유 기능**
- 강의리스트를 다른 사용자에게 공유
- 자동완성 기능 (팀/이름/아이디 입력)
- 공유된 사람 확인 기능
- 공유된 리스트는 "공유 강의리스트" 드롭다운에서 확인

### 4. **플러그인 바 시스템**
- 우상단에 플러그인 아이콘 배치
- 아이콘과 이름 함께 표시
- 드롭다운 메뉴로 기능 접근
- **리스트 관리 플러그인**: 리스트 이름 수정, 강의 추가, 삭제
- **공유 플러그인**: 공유하기, 공유된 사람 확인
- **업무요청 플러그인**: 세금계산서 발행요청
- **운영업무 플러그인**: 세금계산서 발행, 강의료 정산파일 제작

### 5. **자동화 기능**
- 내 강의리스트에서만 사용 가능
- 세금계산서 발행요청
- 세금계산서 발행
- 강의료 정산파일 제작

### 6. **UI/UX 개선**
- 메뉴명 표시 바 (헤더 아래, 현재 선택된 메뉴 표시)
- 모달 오버레이 (배경 딤 처리)
- 선택한 강의 요약 (터미널 스타일, 수평 스크롤)
- 카테고리별 선택 개수 표시
- 날짜 형식: `YYYY-MM-DD HH:mm`
- 회색 톤 색상 적용
