# 기능정의서 (강의리스트)

본 문서는 `LectureMaster`의 **소스코드 기준 실제 동작**을 바탕으로, **기획 관점**에서 기능을 요약 정리합니다.  
작성 기준: “사용자가 무엇을 할 수 있는가 / 어디에서 할 수 있는가(위치)”

---

## 기능정의 (프론트 Depth 기준)

아래 Depth는 **프론트(UI)에서 사용자가 실제로 타고 들어가는 구조** 기준입니다.  
Depth는 길어져도 무방하다고 하셔서, “메뉴 → 화면 → 영역 → 액션/상태”를 **Depth 1 / Depth 2 / Depth 3 ... 컬럼**으로 쪼개서 표현했습니다.

---

## 비고
- 본 기능정의서는 **개발 세부 구현(클래스/함수/서비스 구조)** 설명을 제외하고, **사용자 관점 기능**만 요약했습니다.
- 권한 관련 상세(운영자/관리자/최고관리자)는 기존 문서인 `docs/권한_기능목록_강의리스트.md`를 참고하세요.
